## 작성규칙

- 폴더와 파일명은 영문으로 작성한다.
- 모든 문제는 출저를 밝히고 외부링크를 참조한다.
- 모든 문제는 해답없이 죽이되는 밥이되든 직접 풀어본다.
- 문제는 2일이상 못풀시 다른사람의 해답을 참고한다.
- 문제를 다풀어보고 리뷰를한다. 다른사람의 해답을 같이 참조한다.
- 최대한 많은 문제를 풀어보되 풀이과정은 이해될때까지 작성한다.
